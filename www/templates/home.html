Astronomical Algorithms Implemented

This is my implementation of various astronomical algorithms, mostly from Astronomical Algorithms by Jean Meeus, which is the source of most of the others I found on the web (although not all admit that).

I am using this as a test bed for exploring how to integrate C++, Python and JavaScript. I started with my C++ Coordinates library, and wrapped it in Python which is used to implement astronomical algorithms. I then used Flask, to mix python and html which is then served by mod_wsgi in Apache.
