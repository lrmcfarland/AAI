Astronomical Algorithms Implemented

This is my implementation of various astronomical algorithms, mostly from Astronomical Algorithms by Jean Meeus, which is the source of most of the others I found on the web (although not all admit that).

At the moment, it is mostly a proof of concept of a software Turducken. I started with my C++ Coordinates library, stuffed in Python which is used to implement astronomical algorithms, poured into Flask, garnished with a JQuery datepicker, and served by mod_wsgi in Apache.
